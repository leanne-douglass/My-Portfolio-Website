My Web Projects

A selection of projects I've designed and developed

Goose Run Game

Goose Run! is a fast-paced 2D arcade-style game built with Processing (Java). Guide a goose from the park to university while avoiding traffic. Includes a main menu, score tracking, lives system, and a game-over screen.


This project demonstrates OOP design, collision detection, game state management, and user input handling.


Portfolio Website

A personal portfolio that showcases design and coding skills. Built with semantic HTML5 and modern CSS for an accessible, elegant, and responsive experience.


Features a consistent brand system, responsive layout, and a clean, professional aesthetic.


Online Dog Daycare Service

A front-end landing page demo with navigation, hover effects, and a booking form. Maggie’s Pawcations is a bright, welcoming homepage for a local dog daycare service in Newcastle upon Tyne, using a cheerful yellow and orange colour palette.



The layout includes a clear navigation bar and well-structured sections for services and client reviews, with friendly typography and a playful tone.
